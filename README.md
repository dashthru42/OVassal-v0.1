First Build of OVassal Chess Engine, v0.1, a working project in C++23.

OVassal is a bitboard-oriented chess engine designed to experiment with different implementations
of core chess programming paradigms. It is designed to be completely open-source, (eventually) accessible
once the frame is polished, and to serve as a spine or template for anyone to alter in C++.

Currently the engine is in its early stages, undergoing perft testing and DFS improvement.
The engine is not currently designed for competitive play, but is designed with a competitive
eventuality in mind.
